Privacy Policy

Effective date: May 14, 2026

Voronoi RVE Generator is a PowerPoint add-in for generating 2D biomimetic porous Voronoi RVE images from user-entered parameters.

Data Collection

This add-in does not collect, store, transmit, sell, or share personal data. RVE parameters and generated graphics are processed locally in the user's browser or Office task pane.

Generated Content

Generated SVG or PNG images are created locally and inserted into the active PowerPoint document only when the user chooses to insert them.

External Services

The add-in loads the Microsoft Office JavaScript API from Microsoft's official CDN. The Voronoi generation library is hosted with this add-in's static files.

Contact

For support or privacy questions, use the support page linked from the add-in listing.
