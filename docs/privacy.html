Privacy Policy

Effective date: May 14, 2026

Voronoi RVE Generator is a browser-based tool for generating 2D biomimetic porous Voronoi RVE images from user-entered parameters.

Data Collection

This tool does not collect, store, transmit, sell, or share personal data. RVE parameters and generated graphics are processed locally in the user's browser.

Generated Content

Generated SVG or PNG images are created locally and downloaded only when the user chooses to save them.

External Services

The generator loads the Voronoi generation library in the browser. No host application is required.

Contact

For support or privacy questions, use the support page linked from this project.
