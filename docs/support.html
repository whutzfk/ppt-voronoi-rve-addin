Voronoi RVE Generator is a PowerPoint task pane add-in for creating 2D biomimetic porous Voronoi RVE images.

Basic Use

Open the add-in task pane, set RVE size, pore count, porosity or wall thickness, structural uniformity, variation coefficient, and random seed, then generate a preview. Use the insert button to place the generated image into the active slide.

Troubleshooting

If the add-in does not load, confirm that PowerPoint has internet access and that Office add-ins are enabled by your organization. If insertion fails in a normal browser tab, open the add-in inside PowerPoint; insertion requires the Office host environment.

Contact
