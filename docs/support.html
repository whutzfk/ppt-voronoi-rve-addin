Voronoi RVE Generator is a browser-based tool for creating 2D biomimetic porous Voronoi RVE images.

Basic Use

Open the generator in a browser, set RVE size, pore count, porosity or wall thickness, expansion ratio, structural uniformity, variation coefficient, random seed, and colors, then generate a preview. Use the download buttons to save SVG or PNG output.

Troubleshooting

If the generator does not load, refresh the browser page and confirm JavaScript is enabled. If downloads are blocked, allow downloads from the page in your browser settings.

Contact

For support, report an issue through the project's GitHub repository after it is published.
