Before submitting to Microsoft Partner Center:

1. Create the GitHub repository `whutzfk/ppt-voronoi-rve-addin`.
2. Push this project to the repository.
3. Enable GitHub Pages from `main` and `/docs`.
4. Open the production add-in URL and confirm the task pane loads.
5. Sideload `manifest.production.xml` in PowerPoint and test insertion.
6. Submit `manifest.production.xml` in Partner Center.
